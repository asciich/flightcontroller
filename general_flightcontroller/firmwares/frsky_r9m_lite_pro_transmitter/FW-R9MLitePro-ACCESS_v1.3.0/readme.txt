readme.txt                                                           Release note 
 
Firmware Version: v1.3.0

The release firmware resolved the issues below:
--------------------------------------------------------------------------------------------------------------------
1. Improved RF performance.
2. Added the telemetry for LFR (Lost Frame Rate).
3. Added the telemetry for the power value in self-power adaptive mode.
-------------------------------------------------------------------------------------------------------------------
How to update R9M lite Pro firmware :
By radio (SD card) :
1. Put the firmware under the folder [FIRMWARE] of sd card.
2. Plug in R9M Lite Pro into the module bay of the radio which has Lite style module bay.
3. Power on the radio and find the firmware,select it by press [ENT].
4. Select 'Flash ext. module', wait to end.
