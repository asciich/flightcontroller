# Pixhack

## Output Channel mapping

The following list shows how to map the Pixhack output channels to the ArduPilot output channels:

```
S1  <-> RC1
S2  <-> RC2
S3  <-> RC3
S4  <-> RC4
S5  <-> RC5
S6  <-> RC6
S7  <-> RC7
S8  <-> RC8
FS1 <-> RC9
FS2 <-> RC10
FS3 <-> RC11
FS4 <-> RC12
FS5 <-> RC13
```
