# Default channel configuration

This document sums up my personal default channel configuration.

## Default channel configuration for Betaflight:

```
1 -> Throttle  -> Throttle
2 -> Rudder    -> Yaw
3 -> Elevation -> Pitch
4 -> Aileron   -> Roll
5 -> Arm motors
6 -> Pre arm
7 -> cam and osd selection
8 <Currently not used>
```
